# Print Manzil

**Live link**: [https://job-scope.vercel.app/login](https://job-scope.vercel.app/login)

## Run this project

#### Install all packages

```bash
npm install
# or
bun install
```

#### Create `.env` file with this content

```javascript
VITE_BASE_URL = "https://interview.pencilwoodbd.org";
```

#### Run development server

```bash
npm run dev
# or
bun run dev
```
